== Description ==
BushChang Membership is a simple plugin for displaying either a social login prompt or membership profile.

== Installation ==
Installation is easy.

1. Install BushChang Membership either via the WordPress.org plugin directory, or by uploading the files to your server.
2. After activating BushChang Membership, you will be able to customize the plugin options under the [Settings Discussion Screen](http://codex.wordpress.org/Settings_Discussion_Screen).
3. That’s it. You’re ready to go!
